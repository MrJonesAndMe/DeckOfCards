# DeckOfCards
Images of a deck of cards drawn in Illustrator to use in educational projects.

The naming convention for the deck of cards uses the suit in all lowercase (hearts, diamonds, spades, and clubs) and the the card (2, 3, 4, 5, 6, 7, 8, 9, 10, jack, queen, king, ace) with an underscore between them ending with a .png file extension (ex: hearts_4.png).  You can remove the last 4 characters from the file name and then split it by the underscore to separate the suit and the card.  There are also two jokers and six different card backs as listed below:

back_1.png, back_2.png, back_3.png, back_4.png, back_5.png, back_6.png, joker_black.png, joker_red.png
